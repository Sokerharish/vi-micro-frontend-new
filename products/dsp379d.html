TMS320F28379D Dual Core PWM Controller (Micro–Launch379D)

The Micro–Launch379D is designed with improved performance, enhanced peripheral features, and optimized cost. Built around the LAUNCHXL-F28379D DSP Controller platform, it includes a dedicated carrier board with multiple interfaces — making it suitable for laboratory experiments in power electronics, motor control, electric vehicle technology, and embedded system applications.

Control Algorithms Supported

  • FOC Implementation for PMSM & Induction Motors
  • DTC Implementation for PMSM & Induction Motors
  • Sensorless Implementation for PMSM & Induction Motors

LAUNCHPAD-F28379D Hardware Features

  • USB connected isolated XDS100v2 JTAG debug probe for real-time debug and flash programming
  • 4×20 pin headers/connectors for easy expansion
  • Programmable buttons and LEDs
  • TMS320F28379D: 200 MHz dual C28x CPUs and dual CLAs
  • 1MB Flash, 16-bit or 12-bit ADCs, comparators, 12-bit DACs
  • Delta sigma filters, 16 HRPWMs, eCAPs, 2×eQEPs, 2×CANs
  • Supports two Booster Pack Plug-in modules
  • Two encoder interface connectors

Carrier Board Specifications

  • Isolated CAN transceiver connector
  • 34-pin FRC Header for 12 PWM buffered signals & 2×QEP signal termination
  • 26-pin FRC for 16-channel ADC input termination
  • 2×12-bit DAC outputs at separate connector
  • 4 push button switches, 2 limit switches
  • 4 LEDs
  • CAN BUS terminated at 3-pin RMC Connector
  • Extended I/O: 8 slide switches, 8 LEDs, 20×4 LCD Display
  • One I2C & One UART at 6-pin RMC Connector
  • Linear power supply: 230V / 50Hz, +5V / 2A
  • 4×20 pin socket connector for LAUNCHXL-F28379D Launchpad
  • Enclosed in a sleek plastic cabinet

Applications

  • Power electronics lab experiments (AC/DC converters, inverters)
  • Motor control: PMSM, BLDC, Induction motor drives
  • Electric Vehicle (EV) lab — BMS, motor drive, regenerative braking
  • Embedded control system prototyping and R&D
  • MATLAB/Simulink HIL (Hardware-in-the-Loop) testing
