MR Robo UNO Q — ROS2 Autonomous Robot with Qualcomm Dragonwing™

The MR Robo UNO Q is built around the Arduino UNO Q — a dual-processor development board combining the Qualcomm QRB2210 SoC with an STM32U585 co-processor. It ships with 2× LiDAR sensors, USB camera, and a 30Ah battery, making it the most capable mid-range robot in the Vi Microsystems lineup for full SLAM and autonomous navigation without a GPU.

Dual Processor Architecture

Application Processor

Qualcomm Dragonwing™ QRB2210
  • Quad-core Arm Cortex-A53 @ 2.0 GHz
  • Adreno GPU (3D graphics accelerator)
  • 2× ISP (13 MP + 13 MP / 25 MP) @ 30 fps
  • Linux OS (Ubuntu-based)
  • Runs ROS2 Humble full stack

Real-Time MCU

STM32U585 Cortex-M33
  • Arm Cortex-M33 @ 160 MHz
  • 2MB Flash + 786KB SRAM
  • 3× USART, 4× I²C, 3× SPI, 1× FDCAN
  • RGB LEDs, 8×13 LED Matrix
  • Real-time motor & sensor control

Robot Hardware Components

  • ×2 LiDAR Sensors — 360° real-time distance scanning for SLAM, autonomous navigation, localization, and obstacle avoidance
  • USB Camera Module — Real-time image/video for computer vision, YOLOv8 object detection, and AI-based navigation
  • 30Ah LiFePO4 Battery Pack — Long operational backup, stable voltage, thermal safety, deep-cycle durability
  • BTS7960 Motor Driver Modules ×2 — High-current H-bridge with bidirectional PWM and overcurrent protection
  • 130mm Rugged Rubber Wheels — High-traction for indoor and outdoor terrain
  • High Torque DC Geared Motors — High torque at lower RPM for carrying payloads and rough terrain

What You Can Implement

SLAM + Nav2

Full 2D SLAM with Cartographer, GMapping, SLAM Toolbox. Autonomous navigation, path planning, obstacle avoidance, waypoint following.

Computer Vision

YOLOv8 object detection, person tracking, warehouse navigation, safety monitoring using the USB camera with OpenCV and ROS2.

Sensor Fusion

Fuse wheel odometry, LiDAR, and visual data with robot_localization (EKF/UKF) for stable pose estimation and reduced drift.

Industrial AMR

Build autonomous patrol robots, smart delivery robots, industrial inspection robots, and warehouse AMRs.

Digital Twin

URDF model, Gazebo simulation, Isaac Sim integration. Test navigation algorithms virtually before real-world deployment.

micro-ROS Bridge

STM32U585 handles real-time motor control; QRB2210 runs the full ROS2 stack. Full distributed architecture on one board.
