FOC Implementation for PMSM on TMS320F28379D — From Theory to Code

by Support Team  /  April 2026

Introduction

Field-Oriented Control (FOC) is the gold standard for high-performance PMSM and BLDC motor control. It decouples torque and flux control, enabling precise speed and position response. The TMS320F28379D Micro-Launch379D provides all hardware needed — 16 HRPWMs, dual ADC, encoder interface, and a 200 MHz C28x core — to implement a complete FOC loop.

Step 1 — Clarke Transform (C Code)

foc_clarke.c — Clarke Transformation
// Clarke Transform: 3-phase → 2-phase stationary (α-β)
// Assumes balanced 3-phase: Ia + Ib + Ic = 0

typedef struct {
    float alpha;
    float beta;
} Clarke_t;

Clarke_t Clarke_Transform(float Ia, float Ib) {
    Clarke_t out;
    // α component equals Ia directly
    out.alpha = Ia;
    // β component uses Ib and Ic (=-Ia-Ib)
    out.beta  = (Ia + 2.0f * Ib) * 0.57735f; // 1/√3
    return out;
}

Step 2 — Park Transform (C Code)

foc_park.c — Park Transformation
#include <math.h>

typedef struct { float d; float q; } Park_t;

Park_t Park_Transform(float alpha, float beta,
                        float theta) {
    Park_t out;
    float cos_t = cosf(theta);
    float sin_t = sinf(theta);
    // d-axis (flux component)
    out.d =  alpha * cos_t + beta * sin_t;
    // q-axis (torque component)
    out.q = -alpha * sin_t + beta * cos_t;
    return out;
}

// Inverse Park: rotating → stationary frame
Clarke_t Inv_Park(float Vd, float Vq, float theta) {
    Clarke_t out;
    float cos_t = cosf(theta);
    float sin_t = sinf(theta);
    out.alpha = Vd * cos_t - Vq * sin_t;
    out.beta  = Vd * sin_t + Vq * cos_t;
    return out;
}

Step 3 — PI Current Controller

pi_controller.c — PI Controller for Id and Iq
typedef struct {
    float Kp;           // Proportional gain
    float Ki;           // Integral gain
    float integral;     // Accumulated integral
    float out_max;      // Output saturation limit
} PI_t;

float PI_Run(PI_t *pi, float ref, float fdbk) {
    float err = ref - fdbk;
    pi->integral += pi->Ki * err;

    // Anti-windup clamp
    if      (pi->integral >  pi->out_max) pi->integral =  pi->out_max;
    else if (pi->integral < -pi->out_max) pi->integral = -pi->out_max;

    float out = pi->Kp * err + pi->integral;

    // Output saturation
    if      (out >  pi->out_max) out =  pi->out_max;
    else if (out < -pi->out_max) out = -pi->out_max;

    return out;
}

Step 4 — Main FOC Loop on F28379D

This ISR runs at 10 kHz (100 µs) triggered by the PWM period interrupt on the TMS320F28379D. The dual C28x cores allow the FOC loop on CPU1 and CAN/UART communication on CPU2 simultaneously.
foc_main.c — FOC Interrupt Service Routine
// PWM Period ISR — runs at 10 kHz
__interrupt void pwmISR(void) {
    // 1. Read phase currents via ADC
    float Ia = ADC_ReadCurrent(ADC_CH_A);
    float Ib = ADC_ReadCurrent(ADC_CH_B);

    // 2. Read rotor angle from eQEP encoder
    float theta = eQEP_GetAngle();

    // 3. Clarke Transform: 3-phase → α-β
    Clarke_t clark = Clarke_Transform(Ia, Ib);

    // 4. Park Transform: α-β → d-q
    Park_t park = Park_Transform(clark.alpha,
                                   clark.beta, theta);

    // 5. PI Controllers for Id and Iq
    float Vd = PI_Run(&pi_d, Id_ref, park.d);
    float Vq = PI_Run(&pi_q, Iq_ref, park.q);

    // 6. Inverse Park: d-q → α-β voltages
    Clarke_t volt = Inv_Park(Vd, Vq, theta);

    // 7. SVPWM — generate PWM duty cycles
    SVPWM_Update(volt.alpha, volt.beta);

    // 8. Update HRPWM compare registers
    HRPWM_SetDuty(pwmA, pwmB, pwmC);

    PieCtrlRegs.PIEACK.all = PIEACK_GROUP3;
}
Tune PI gains starting with Kp=0.1, Ki=0.01. Increase Kp gradually until the current response is fast without oscillation. Then increase Ki to eliminate steady-state error. Use CCS Watch expressions to monitor Id and Iq in real time.

Conclusion

You have implemented a complete FOC loop for PMSM on the TMS320F28379D including Clarke transform, Park transform, dual PI current controllers, inverse Park, and SVPWM generation — all running at 10 kHz in the PWM ISR. This forms the basis for EV motor drive, servo control, and industrial automation applications.
