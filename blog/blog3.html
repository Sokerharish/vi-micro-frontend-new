Working Principle — AES

AES is a symmetric block cipher — the same key is used for both encryption and decryption. It operates on 128-bit blocks of data and supports key sizes of 128, 192, or 256 bits. AES performs multiple rounds of substitution, permutation, mixing, and key addition operations to produce ciphertext.

  • SubBytes — Non-linear byte substitution using S-Box lookup table
  • ShiftRows — Cyclic shift of each row in the state matrix
  • MixColumns — Matrix multiplication in GF(2⁸) field
  • AddRoundKey — XOR with the round key derived from the original key

Experiment 22 — AES Encryption in Python

Run this on the Vi LaN-05 station PC. Install pycryptodome: pip install pycryptodome
aes_experiment.py — AES Encrypt & Decrypt
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad
from Crypto.Random import get_random_bytes
import binascii

# ===== AES-256 ENCRYPTION =====
key = get_random_bytes(32)    # 256-bit key
iv  = get_random_bytes(16)    # Initialization Vector

plaintext = b"Vi Microsystems AES Experiment 22"

# ENCRYPT
cipher_enc = AES.new(key, AES.MODE_CBC, iv)
ciphertext  = cipher_enc.encrypt(pad(plaintext, AES.block_size))

print(f"Plaintext  : {plaintext.decode()}")
print(f"Key (hex)  : {binascii.hexlify(key).decode()}")
print(f"IV  (hex)  : {binascii.hexlify(iv).decode()}")
print(f"Ciphertext : {binascii.hexlify(ciphertext).decode()}")

# DECRYPT
cipher_dec = AES.new(key, AES.MODE_CBC, iv)
decrypted  = unpad(cipher_dec.decrypt(ciphertext), AES.block_size)

print(f"Decrypted  : {decrypted.decode()}")
print(f"Match      : {plaintext == decrypted}")

Working Principle — RSA

RSA is an asymmetric cryptosystem using a public-private key pair. A message encrypted with the public key can only be decrypted with the private key. RSA security relies on the computational difficulty of factoring large prime numbers.

  • Choose two large primes p and q
  • Compute n = p × q (modulus) and φ(n) = (p-1)(q-1)
  • Choose public exponent e such that gcd(e, φ(n)) = 1
  • Compute private key d such that d × e ≡ 1 (mod φ(n))
  • Encrypt: C = M^e mod n  |  Decrypt: M = C^d mod n

Experiment 24 — RSA Encryption in Python

rsa_experiment.py — RSA Encrypt & Decrypt
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP
import binascii

# ===== GENERATE RSA 2048-bit KEY PAIR =====
key     = RSA.generate(2048)
pub_key = key.publickey()

print("Public Key:")
print(pub_key.export_key().decode())

# ===== ENCRYPT WITH PUBLIC KEY =====
plaintext  = b"Vi LaN-05 RSA Experiment 24"
cipher_rsa = PKCS1_OAEP.new(pub_key)
ciphertext = cipher_rsa.encrypt(plaintext)

print(f"\nPlaintext  : {plaintext.decode()}")
print(f"Ciphertext : {binascii.hexlify(ciphertext).decode()[:60]}...")

# ===== DECRYPT WITH PRIVATE KEY =====
decipher   = PKCS1_OAEP.new(key)
decrypted  = decipher.decrypt(ciphertext)

print(f"Decrypted  : {decrypted.decode()}")
print(f"Match      : {plaintext == decrypted}")

Experiment 25 — Diffie-Hellman Key Exchange

diffie_hellman.py — Key Exchange Simulation
# Public parameters (known to both parties)
p = 23    # Large prime number
g = 5     # Primitive root modulo p

# Private keys (secret)
a = 6     # Alice's private key
b = 15    # Bob's private key

# Public keys (exchanged over network)
A = (g ** a) % p   # Alice sends A to Bob
B = (g ** b) % p   # Bob sends B to Alice

# Shared secret (computed independently)
secret_alice = (B ** a) % p
secret_bob   = (A ** b) % p

print(f"Alice Public Key : {A}")
print(f"Bob   Public Key : {B}")
print(f"Alice Shared     : {secret_alice}")
print(f"Bob   Shared     : {secret_bob}")
print(f"Keys Match       : {secret_alice == secret_bob}")

Expected Output

When all three experiments run correctly, you will see: AES plaintext matching decrypted output, RSA ciphertext (256 bytes) successfully decrypted, and Diffie-Hellman shared keys matching on both sides — confirming secure key exchange.
